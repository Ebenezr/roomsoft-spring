# Roomsoft Web Server

# Technology stack:
- Spring Boot 3;
- Spring Web;
- Spring Data;
- PostgreSQL database;
- Hibernate;
- Lombok;
- Spring Security (as basic authentication).

## To run this application use:

```bash
mvn spring-boot:run
```

## The view in the Postman:
[http://localhost:8080/api/users](http://localhost:8080/api/users)
